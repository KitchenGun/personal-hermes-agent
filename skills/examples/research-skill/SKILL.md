# Research Skill

## Purpose

Collect and summarize public information for a user-approved research topic.

## When to use

- Trend digests
- Public documentation review
- Source-backed comparison notes

## Inputs

- Topic or question
- Approved source list or search constraints
- Desired output format

## Procedure

1. Clarify the research question if needed.
2. Search public sources only.
3. Prefer primary sources and cite links.
4. Deduplicate repeated claims.
5. Summarize findings, uncertainty, and follow-up questions.

## Outputs

- Markdown summary
- Source list
- Optional next-action checklist

## Safety

Do not collect private data, credentials, private messages, or unapproved personal identifiers.
