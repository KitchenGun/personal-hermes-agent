# Skills

Skill은 Hermes가 반복 업무를 안정적으로 수행하기 위한 절차 단위입니다.

## 예시

- `examples/research-skill/SKILL.md`
- `examples/code-review-skill/SKILL.md`
- `examples/reuse-first-implementation/SKILL.md`
- `examples/skill-optimization-review/SKILL.md`

## 권장 섹션

- Purpose
- When to use
- Inputs
- Procedure
- Outputs
- Safety

Skill 문서에는 실제 endpoint, token, private memory를 넣지 않습니다.
