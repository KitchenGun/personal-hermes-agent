# 00. 포트폴리오 의도

`personal-hermes-agent`는 실제 개인 운영 환경을 그대로 공개하는 저장소가 아니라, 운영 철학과 구조를 설명하는 공개 템플릿입니다.

## 공개 대상

- 아키텍처 설명
- sanitized 설정 예시
- 작업(job) 패턴
- 프롬프트/스킬 설계 패턴
- 보안 체크리스트와 배포 전 검수 절차

## 비공개 대상

- 실제 `.env`, API key, OAuth token, cookie
- 개인 대화 로그, 세션, 장기 메모리 원문
- gateway state, DB, 캐시, 음성/이미지 원본, 개인 경로
- 운영 중인 provider 계정명/조직명/프로젝트 ID
