# 00. Overview

이 저장소는 개인 Hermes Agent 운영 방식을 공개 가능한 형태로 설명하는 sanitized profile입니다.

## 원칙

1. 실제 운영 데이터는 복사하지 않습니다.
2. 모든 설정은 placeholder 또는 synthetic 예시입니다.
3. Job은 Hermes와 대화해 생성/수정되는 Registry 항목으로 관리합니다.
4. 문서는 Memory, Skills, Tools, Gateway, Cron, Delegation, Provider Routing의 연결을 설명합니다.

## 성장하는 개인 AI 에이전트

Hermes는 매일의 대화, 자동화 Job, 리서치, 코드 작업, 자기 점검 결과를 바탕으로 더 나은 운영 루틴을 제안합니다. 단, 장기 Memory 반영은 사용자의 승인과 보안 점검을 거친 후 수행합니다.
