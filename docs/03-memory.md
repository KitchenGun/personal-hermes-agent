# 03. Memory

Memory는 장기적으로 유용한 선호, 반복 맥락, 프로젝트 사실을 저장하는 영역입니다.

## 공개 저장소 원칙

- 실제 memory 원문은 절대 포함하지 않습니다.
- 예시는 synthetic 문장만 사용합니다.
- memory candidate는 사용자 승인 전까지 확정 Memory가 아닙니다.

## 흐름

1. Job 또는 대화 결과에서 후보를 추출합니다.
2. 민감 정보와 일시적 정보는 제거합니다.
3. 사용자가 승인하면 운영 환경의 안전한 memory store에 반영합니다.
4. 공개 repo에는 추출 규칙과 prompt만 남깁니다.
